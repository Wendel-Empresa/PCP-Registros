Como configurar o Google Sheets
PASSO A PASSO
Passo 1 — Criar a planilha
1. Acesse sheets.google.com
2. Crie uma planilha nova
3. Renomeie para PCP Registros
4. Crie estas abas (clicando no + no rodapé):
  • Polietileno
  • Fibra
  • Pias_Tanques
  • Micronizador
  • Gas
  • Carregamento
Passo 2 — Criar o Apps Script
1. Na planilha, clique em Extensões → Apps Script
2. Apague o código que aparece
3. Cole o código abaixo (toque para copiar)
▶ Toque para copiar o código
function doPost(e){try{var d=JSON.parse(e.postData.contents);var ss=SpreadsheetApp.getActiveSpreadsheet();var sh=ss.getSheetByName(d.setor)||ss.insertSheet(d.setor);var keys=Object.keys(d);if(sh.getLastRow()===0){sh.appendRow(keys);}sh.appendRow(keys.map(function(k){return d[k];}));return ContentService.createTextOutput(JSON.stringify({ok:true})).setMimeType(ContentService.MimeType.JSON);}catch(err){return ContentService.createTextOutput(JSON.stringify({ok:false,error:err.toString()})).setMimeType(ContentService.MimeType.JSON);}}
Passo 3 — Publicar o script
1. Clique em Implantar → Nova implantação
2. Em "Tipo", selecione App da Web
3. Em "Executar como", selecione Eu
4. Em "Quem tem acesso", selecione Qualquer pessoa
5. Clique Implantar
6. Autorize quando pedir permissão
7. Copie a URL que aparece no final
Passo 4 — Colar a URL no app
1. Volte para esta tela inicial
2. Cole a URL no campo "Conexão com Google Sheets"
3. Clique Salvar configuração
4. O indicador ficará verde ✅

A partir daí, cada registro enviado pelos encarregados aparece instantaneamente na sua planilha!
